# Аналіз відкритих даних: Спортивні звання України (2023)

### Мета проєкту
Метою цієї роботи є проведення аналітичного дослідження відкритих даних щодо присвоєння спортивних звань в Україні протягом 2023 року. Проєкт спрямований на виявлення закономірностей у розвитку спорту, аналіз гендерного розподілу та інклюзивності у спортивній сфері.

### Джерело відкритих даних
Дані отримані з Єдиного державного порталу відкритих даних (data.gov.ua):
[Статистичні дані щодо кількості осіб, яким присвоєно спортивні звання](https://data.gov.ua/dataset/sport-titles/resource/91eddf84-eade-4a1b-8e72-8f48092426b6)

### Питання та гіпотези для аналізу
1. **Гендерний аналіз:** Яка частка жінок (`sex2`) та чоловіків (`sex1`) серед усіх отриманих звань за рік? Гіпотеза: кількість чоловіків переважає у загальній статистиці.
2. **Динаміка за періодами:** У які місяці спостерігається найбільша активність присвоєння звань? Гіпотеза: кількість нагороджень зростає наприкінці кварталів або року.
3. **Аналіз інклюзивності:** Який відсоток від загальної кількості звань складають особи з інвалідністю (`invalidity`)? Гіпотеза: частка таких звань залишається стабільною протягом року без різких стрибків.
